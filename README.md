# Day-36-Docker-Project-Dockerize-a-Full-Application


> On Day 36, I dockerized a full application using Docker.
> I created a simple Flask app and wrote a Dockerfile using a lightweight base image.
> I ensured security by running the container with a non-root user.
> I used Docker Compose to manage the app and a PostgreSQL database.
> I added volumes for data persistence and used a .env file for configuration.
> I also implemented a health check for the database service.
> Finally, I pushed the image to Docker Hub and tested it in a fresh environment successfully.
